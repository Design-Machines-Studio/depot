# Shop

Build a shopping list from recipes or a meal plan, organized by the best Bali store for each ingredient, with product links and the option to push to Apple Reminders.

## Process

### Step 1: Gather Ingredients

Determine the source of ingredients:

- **From a meal plan:** Extract all ingredients from the plan generated by `/meal-plan`.
- **From scheduled meals:** Use `get_scheduled_ingredients(days)` to pull ingredients from meals already scheduled in Apple Calendar.
- **From specific recipes:** User names recipes -- pull from Mela via `search_recipes`/`get_recipe` or from pasted/described recipes.
- **From a raw list:** User provides ingredient names directly.

Consolidate: combine duplicate ingredients, sum quantities, note which recipe each ingredient serves.

### Step 2: Load Sourcing Guide

Read `plugins/chef/skills/cooking/references/bali-sourcing.md` for store details, URLs, and specialties.

### Step 3: Route to Stores

For each ingredient, determine the best store based on:

1. **Quality alignment with Eve's framework** -- organic, grass-fed, wild-caught, sustainably sourced preferred.
2. **Availability** -- is the ingredient actually stocked?
3. **Practicality** -- minimize the number of separate orders. Consolidate where possible.
4. **Price** -- note when a budget option exists at a different store.

**Store routing priority by category:**

| Category | Primary Store | Alternative |
|----------|--------------|-------------|
| Fresh produce/herbs | Local markets, Island Organics | Bali Organic |
| Seafood (sustainable) | Bali Sustainable Seafood | Benoa Fish Market (budget) |
| Quality meat cuts | Local Parts by Locavore | Wanaprasta (grass-fed) |
| Bulk protein | Hunters Bali | Bali Organic |
| Organic chicken/eggs | Wanaprasta | Bali Organic |
| Pantry staples | Bali Direct Store | Bali Organic |
| Nuts/seeds/grains | Bali Direct Store | MaiMai Natural Food |
| Specialty bread | Bali Buda | -- |
| Dark chocolate | HAS Chocolate | Bali Direct Store |
| Western/imported items | Pepitos | Bintang |
| Tempeh/tofu (fresh) | Local markets | Bali Organic |

### Step 4: Build the List

Format store by store:

```
## Shopping List for [date range or recipe names]

### Local Markets (Ubud)
- [ ] Ginger, 1 knob
- [ ] Fresh turmeric, 50g
- [ ] Tempeh, 2 blocks
- [ ] Free-range eggs, 1 dozen

### Bali Sustainable Seafood
**URL:** https://shop.balisustainableseafood.com
- [ ] Wild-caught tuna, 500g
- [ ] Prawns, 400g

### Bali Direct Store
**URL:** https://balidirectstore.com
- [ ] Brown rice, 1kg
- [ ] Walnuts, 200g
- [ ] Extra-virgin olive oil, 500ml

### Not Available Locally
- [ ] [Item] -- **Substitute:** [alternative with explanation]

### Estimated Order Count: [N] stores
```

### Step 5: Push to Reminders (Optional)

Ask if Travis wants the list pushed to Apple Reminders:

- Use `add_grocery_items(items, list_name)` to add items.
- Default list: "Groceries" (configurable via MELA_GROCERY_LIST env var).
- Group by store in the list for easy shopping.
- Only push if the user confirms.

### Step 6: Flag Substitutions

For any ingredient not available in Bali:

- Suggest the closest local substitute.
- Explain how the substitute affects the recipe (flavor, texture, cooking behavior).
- Note if the substitution changes the Eve-alignment rating.
